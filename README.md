# Backend_EmployeManagmentSystem
# Employe All Details

This is a backend Employe Management System project developed using Java, JDBC, Servlet, MySQL, HTML, CSS, and JavaScript.

## Features

- Employe Add Operation
- Employe Update Operation
- Employe Delete Operation
- Employe View Details
- Databas Connectivity using JDBC

## Technologies Used

- Java
- JDBC
- Servlet
- MySQL
- HTML
- CSS
- JavaScript
